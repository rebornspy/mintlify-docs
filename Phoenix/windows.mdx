## Window Creation

`Phoenix:CreateWindow()` is the method used to create everything needed for UI development with Phoenix. Call it once after initializing the library.

```lua
local Window = Phoenix:CreateWindow({
    Name = "Example Window",
    BootTitle = "Boot Window",
    MainTitle = "Main Window",
    Theme = "Default"
})
```
