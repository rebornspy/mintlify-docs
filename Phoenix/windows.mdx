## Window Creation

`Phoenix:CreateWindow()` is the method used to create everything needed for UI development with Phoenix. Call it once after initializing the library.

```lua
local Window = Phoenix:CreateWindow({
    Name = "Example Window",
    BootTitle = "Boot Window",
    MainTitle = "Main Window",
    Theme = "Default"
})
```

### Options

<ResponseField name="Name" type="string" required>
    If no boot/main title names are provided, this will take over.
</ResponseField>

<ResponseField name="BootTitle" type="string">
  Name displayed at the top of the boot window. If none is provided, the "Name" option will take over.
</ResponseField>

<ResponseField name="MainTitle" type="string">
  Name displayed at the top of the main window. If none is provided, the "Name" option will take over.
</ResponseField>

<ResponseField name="Theme" type="string">
  Theme of the entire UI. Defaults to "Default" if no theme is provided. This can be changed while the script is running.
</ResponseField>
